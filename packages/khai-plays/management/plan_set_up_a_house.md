# Plan: Set Up a House

## Taxonomy

A **mandate**: the Roadie's inbound checklist, run when a house is raised or has
fallen behind the blueprint.

## Owner

- Owner: [The Roadie](position_roadie.md)

## Direction

A house the chain raises is stocked and current, so the Playwright can produce
against the full stack and the house is green the instant it is stamped.

## Orders

The Roadie runs `khai-stage` and refreshes on change. It moves the work, never
authors the play or changes the rules; raising a new house is the Choregos's
call, the Roadie stocks the one that exists.

## Implementation

`khai-stage` and the blueprint; the registry; the engines the house uses.
Deterministic stamping, not improvisation.

## Targets

- [ ] the stack is stocked: canon, the spine contract, the engines the house uses
- [ ] the management structure is stamped: voice layer, Discussion Standard, the company, the discussions and orders homes
- [ ] each engine the house uses is materialized into the repo
- [ ] the house registry is emitted, so the house is green on raise
- [ ] the house conforms (`khai-tests --project`)
