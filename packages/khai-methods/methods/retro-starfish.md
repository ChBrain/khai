  - key: keep
    label: Keep doing
    question: "What should we keep doing (it is working well)?"
  - key: less_of
    label: Less of
    question: "What should we do less of (it is not adding enough value)?"
  - key: more_of
    label: More of
    question: "What should we do more of (it has potential but is underused)?"
  - key: start
    label: Start doing
    question: "What should we start doing (we are not doing it yet but should)?"
  - key: stop
    label: Stop doing
    question: "What should we stop doing (it is not adding value)?"
---

A five-point retrospective format shaped like a starfish: each arm represents a different action stance toward current practices. The format distinguishes between **more of** (underused practices worth amplifying) and **start** (new practices not yet tried), giving finer resolution than a simple keep/change split.

Introduced by Pat Kua in 2006. Published at thekua.com and included in his _Retrospective Coaching Cards_ deck.
