The authoring method: cut to fit, one element per load-bearing vector. Name the single force each element carries, remove anything that carries none or repeats one already carried, and tighten each to the smallest form that still holds its vector. The cast reaches whatever floor its kind needs to do its work, judged not counted: a debate wants several voices, a run wants its beats, but the number is the house's and a thin cast is a weakness to weigh, never a gate to trip.

This is khai's own craft, the human rung of the ladder written down (see `docs/BOUNDARY.md`): guidance a person applies, with the deterministic walls beneath as the backstop and never a substitute for taste. It is where the cut-to-fit floors live, as authoring judgement rather than as a computed rule, because what makes an element load-bearing is meaning, which no wall can weigh.
