# Volition

Volition as composite: the crossing from wanting to doing that Heckhausen and Gollwitzer mapped as the Rubicon -- a goal deliberated (the wish), an if-then plan that carries it across into action (the will), and the executive control that holds the pursuit against interference (the doing) -- and the drama lives where the crossing fails, the wish that never becomes a will and the will that cannot be carried out.
