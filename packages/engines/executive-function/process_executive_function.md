# Process: Executive Function

## Taxonomy

Cognitive Control.

## Owner

- Project: khai
- Engine: executive-function

## Initiated by

The persona has a goal that requires more than one step, a competing impulse that must be blocked before it fires, or a shifting situation that requires releasing the current configuration and loading another.
Any moment the automatic response is not the correct one -- or the path from here to the goal is not already laid -- the regulatory layer engages.

## Direction

Toward the goal, against the interference.
The persona holds the target active, suppresses what would derail it, updates what is held in working memory as the task shifts, and sequences the intermediate steps in order.
The direction is not toward an object in the world but toward the conditions under which intentional action can happen at all.

## Lever

The goal meets the weight of competing pulls.
The prepotent response is available, the context is shifting, working memory is under load, and the path to the goal is not self-evident.
The regulatory layer does not remove the competition; it holds the goal against it.
Where the competition is weak or the goal is automatic, the layer runs lightly; where the competition is strong or the task is novel, it is expensive and can fail under load.

## Echo

The goal-directed sequence runs, stalls, or is lost.
The persona either holds the line or the interference takes it -- the wrong answer fires, the wrong task runs, the plan dissolves into its parts.
The process continues as one of [inhibition](process_inhibition.md), [updating](process_updating.md), [shifting](process_shifting.md), or [planning](process_planning.md), each a distinct mode of regulatory cost the moment requires.
When the goal changes or the task demands shift, the layer re-engages here.
