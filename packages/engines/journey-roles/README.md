# Journey-roles

Journey roles as persona: the recurring character functions of the hero's journey -- hero, mentor, herald, threshold guardian, shapeshifter, shadow, ally, and trickster -- filled by whichever character the story needs to fill it, for as long as it needs the function.

This engine is defined by its [manifest](package.json), which the canon renders as the WIRES card. The manifest is the single source of truth; this README is generated, not edited by hand.
