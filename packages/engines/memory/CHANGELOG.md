### Patch Changes

- b5ba349: engine: add the memory engine (new khai-original engine, not a Cultures lift). Memory as process -- how the past re-enters the present: a cue meets a trace, the persona rebuilds (not replays) what was, and refiles it changed. A process tree (root + five kinds): episodic, semantic, procedural, emotional, prospective. Grounded in memory science (Bartlett's reconstruction, Tulving's episodic/semantic, Squire's procedural, Nader's reconsolidation, Loftus's misinformation effect). Complementary to the inner-layer engines emotion/attachment/motivation; the present feeling is delegated to the emotion engine, forgetting is the root's Echo inherited by every kind. Atomic: no cross-engine references in content.
