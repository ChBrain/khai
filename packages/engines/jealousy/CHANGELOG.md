# @chbrain/khai-engine-jealousy

## 0.1.1

### Patch Changes

- 979d494: Add the jealousy engine: a rival, real or imagined, threatening a valued bond the persona already holds -- one anchor process (jealousy), one process appraising the threat (threat appraisal), one position holding the stance (jealous), and one process for the guarding it drives (guarding). Grounded in Parrott & Smith (1993) and Buss (2000), kept distinct from the two-party envy engine.
