# Process: Adjourn Group

## Taxonomy

[Evolve Group](process_evolve_group.md)

## Owner

- Project: khai
- Engine: group

## Initiated by

The shared context that held the group together is ending.
The project closes. The class graduates. The household separates.
What was the reason for being in the same room is no longer the reason.

## Direction

Toward separation.
Each person carries out of the room what the group made of them while they were in it.
What was shared becomes individual, and the room loses what kept the people in it.

## Lever

The pull of the next room each person is going to.
The group is still here, but each person is already partly somewhere else.
What is said in this final phase is said into a thinning room.

## Echo

The group ends, but not all at once.
Names are still mentioned for a while; in-jokes still land; the language the group built persists in each person's vocabulary.
It does not end the moment the room empties. It ends when the last person who carried its pattern lets go of it.
If the shared context comes back -- a reunion, a re-formed project -- the pattern can [perform](process_perform_group.md) again, briefly, before this stage returns.
