# Position: Shared-Information Bias

## Taxonomy

[Belonging, Consensus](position_belonging_consensus.md)

## Owner

- Project: khai
- Engine: bias

## Has

The flashlights swing back to the same patch of grass because three beams already crossed there, and swinging toward it again feels like progress. The one beam that swept past the fence line, where something moved, never gets asked back.

## Orders

Bring the beam back to where the light's already thick. Ask what everyone saw, nod at it, retell it once more, and count the fence line as covered because nobody has mentioned it twice.

## Loses

The one report that would turn the search around. It sits in a single memory, never echoed, so it never picks up the weight that repetition hands to everything else, and the party moves on without it.

## Drives

On the loss of the one report that would turn the search around:
leans into the tilt where it walks the same lit patch all night, mistaking a fact said four times for a fact confirmed four times,
or corrects against it and pays for stopping the whole line to make the quiet one repeat the sighting until it's heard as real.
