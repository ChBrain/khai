# Position: Dunning-Kruger

## Taxonomy

[Self, Competence](position_self_competence.md)

## Owner

- Project: khai
- Engine: bias

## Has

One skill doing double duty.
The competence that would do the task well is the same competence that would notice it done badly, so where the skill runs thin the persona has no way to see that it does. The less there is to work with, the more finished the work looks from the inside.

## Orders

Call it done the moment it feels done.
Read a correction as the other person's mistake, not a signal about the work; speak with most certainty exactly where the ground is weakest, because nothing inside reports the weakness.

## Loses

The second sight that would catch the error.
It is built from the very skill that is missing, so it fails just where it is most needed; the sense that something might be wrong goes quiet precisely where the wrongness lives, and a correction lands as an insult.

## Drives

On the loss of the second sight:
leans into the tilt where the claim keeps outrunning the result and every correction reads as an attack,
or corrects against it and pays in a doubt that will not switch off, second-guessing even the skill that has since become real.
