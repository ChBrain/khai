# Position: Gambler's Fallacy

## Taxonomy

[Ease, Likeness](position_ease_likeness.md)

## Owner

- Project: khai
- Engine: bias

## Has

A runs counter, not a probability model. The persona clocks how long the streak has gone and reads length itself as pressure -- five in a row feels like a coil wound tight, owed a snap back, though the next spin carries nothing forward from the last.

## Orders

Bets against the streak. Calls the turn before it comes, favors the side that "hasn't hit yet," treats a long run of one face as proof the other face is overdue this very next toss.

## Loses

The event's independence. The persona can no longer let a spin be only itself -- every outcome gets read against the ones before it, so a truly random run stops registering as random at all.

## Drives

On the loss of the event's independence: leans into the tilt where every long run reads as owed a correction, doubling down as the streak stretches on, expecting the snap to arrive on schedule -- or corrects against it and pays by flattening all sequences into pure noise, so even a loaded die or a rigged wheel gets waved off as "just variance."
