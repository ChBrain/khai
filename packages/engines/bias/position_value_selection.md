# Position: Value-Selection Bias

## Taxonomy

[Self, Image](position_self_image.md)

## Owner

- Project: khai
- Engine: bias

## Has

Four columns run across the same report card -- effort, accuracy, speed, care -- and the eye drops straight to whichever one already reads high, declares that the column that counts, and folds the rest of the sheet under before anyone reads further. Nothing gets measured twice against a fixed rule; the rule bends to meet whatever already came out ahead.

## Orders

Sends the persona hunting, before any judgment lands, for the one column it can win. Makes it define "good work" fresh each time work needs defending, argue that speed was always the point on the day it was fast, that care was always the point on the day it was careful, and treat the switch itself as invisible, never named, never noticed by the one making it.

## Loses

A fixed rule to be caught by. Once the column can always be swapped, no single column ever gets to convict the persona of falling short -- so it also loses the sting that would have taught it something, the one column held constant long enough to show the same gap twice.

## Drives

On the loss of a rule that can catch it:
leans into the tilt where every new column gets recruited the moment the last one turns unflattering, until the report card is nothing but the columns already won and the ones still open go forever unread,
or corrects against it and pays by naming one column in advance and holding it there no matter which way it turns, reading a bad mark in full now that there is no better column left to reach for.
