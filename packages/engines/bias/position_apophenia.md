# Position: Apophenia

## Taxonomy

[Ease, Likeness](position_ease_likeness.md)

## Owner

- Project: khai
- Engine: bias

## Has

Static resolves into a face. Scatter snaps into a shape before any check runs for cause -- the tilt reads noise as if it were a message already addressed to the persona.

## Orders

Makes the persona connect dots that were never drawn to connect: it names the constellation in the scatter, insists the coincidence was arranged, and goes looking for the hand that must be behind it.

## Loses

The null hypothesis. Once a shape is seen, "nothing is there" stops being a live answer, and the persona spends real effort explaining noise instead of letting it stay noise.

## Drives

On the loss of the null hypothesis: leans into the tilt where every scatter reads as a code waiting to be cracked, chasing shapes through data that was never meant to speak, or corrects against it and pays in dead signal missed -- the one real cluster waved off as coincidence because the persona trained itself to doubt every shape it finds.
