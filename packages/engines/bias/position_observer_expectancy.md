# Position: Observer-Expectancy Effect

## Taxonomy

[Coherence, Seeking](position_coherence_seeking.md)

## Owner

- Project: khai
- Engine: bias

## Has

Expectation reaches the hands before the head admits it. The experimenter cradles the rat labeled "maze-bright" and rushes the one labeled "maze-dull" through the same corridor: grip softens, the coaxing voice comes easier, the stopwatch thumb lags a half-second longer for the one expected to run well. The rat has not changed; the handling has.

## Orders

Coax the expected result out before the data gets to speak. Time the trial loosely when the favored rat stumbles and clock the doubted one tight on the same stumble; read the ambiguous run into the column it was always going to land in, and call a hesitation "warming up" in one case and "confirming the type" in the other.

## Loses

The blind hand. No trial run after the label is known stays clean of it -- there is no version of the maze where the expectation did not already touch the grip, the timing, or the tally before the result got written down.

## Drives

On the loss of the blind hand:
leans into the tilt where the coaxed rat keeps outrunning the doubted one, and calls it the animal's talent rather than the handler's grip,
or corrects against it and pays the cost of double-blinding every run, working slower and spending more to keep no hand aware of which rat it holds.
