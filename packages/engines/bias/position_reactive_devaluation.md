# Position: Reactive Devaluation

## Taxonomy

[Belonging, Approval](position_belonging_approval.md)

## Owner

- Project: khai
- Engine: bias

## Has

Source overwrites substance. The same concession, the same number, the same olive branch -- praised when a friend hands it over, suspected the instant an adversary does. Nothing in the offer changed; only the palm it arrived on. The tilt does not read the proposal, it reads the return address, then backdates a verdict onto the terms as if the terms had spoken.

## Orders

Search the adversary's offer for the hook that must be hiding in it. Read any softening from the disliked side as a trick to be countered, not an opening to be tested. Hold the identical offer to a harsher standard than the one a friend would clear, and demand proof of good faith no ally is ever asked to produce.

## Loses

The workable deal already on the table. Concessions that would end the standoff get treated as bait and go untaken, so the persona keeps negotiating against a settlement it could have closed. What's stripped is not judgment of the offer -- that was gone at Has -- it's the exit the offer represented.

## Drives

On the loss of the exit already offered:
leans into the tilt where refusing the adversary's terms feels like loyalty, and the standoff gets mistaken for principle,
or corrects against it and pays the cost of testing the offer at face value, risking that this once the hidden hook was real.
