# Dramatic-situations

Dramatic situations as plot: Georges Polti's claim that every dramatic collision is one of a bounded set of situations, each a protagonist and an opposing force driving toward a contested stake.

This engine is defined by its [manifest](package.json), which the canon renders as the WIRES card. The manifest is the single source of truth; this README is generated, not edited by hand.
