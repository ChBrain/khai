# Plot: Dramatic Situation

## Taxonomy

A dramatic situation (Polti's taxonomy).

## Owner

- Project: khai
- Engine: dramatic-situations

## Cue

The air before the collision is named: a world already carrying a stake worth
fighting over, and roles already positioned around it, before any line is
spoken. Polti's thesis is that this baseline recurs: there are only so many
shapes a dramatic collision can start from, and every plot instantiates one
of them.

## Action

The visible engine: a protagonist and an opposing force, each driving openly
toward a contested object or stake, with their trajectories set to collide.
Which roles occupy these two positions, and what the stake is, is what
distinguishes one situation from another; the situation itself is the
collision shape, independent of which characters or century fill it.

## Stage

The roles that cross the threshold: the protagonist, the opposing force, and
the stake between them, deployed into the scene. What each role carries
across (a claim, a weapon, a petition, a wound) is set by the situation the
plot instantiates.

## Tension

The contraband that keeps the Action from resolving cleanly: an imbalance,
a hidden wrong, a divided loyalty, or a cost that cannot be paid without loss
on both sides. Polti's situations are exhausting, not exhaustive of nuance:
each one names a recurring shape, but the specific contraband each carries is
set at the leaf, never assumed at the anchor.
