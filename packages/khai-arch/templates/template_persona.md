# Persona: [title]

## Taxonomy

The group above this persona: the role it fills, its seat in the cast or the org. Name it, then ask whether that role already exists as a Position with its own file. If it does, link it; if it does not, ask whether it should. This is what the persona is one of, not its own name (the H1 carries that) and not where it came from (Owner carries that).

## Projection

How the character shows itself to the room and to others: the face it controls and presents, physical and present tense. It must bite the Shadow, the part it cannot control: if what it shows and what it hides do not conflict, there is no persona, only a label. Link the positions and pieces it carries where they have a file; where they do not, ask whether they should use a khai type.

## Action

What the character does: at least one move, just enough (less is more), each producing a cue the session can answer. Deliberate and physical, it must stay distinct from the Tell: if the chosen move and the involuntary leak read the same, nothing is hidden. Link what it touches where that has a file; where it does not, ask whether it should use a khai type.

## Shadow

What the character cannot see in itself and does not control, in conflict with the Projection: at least one, just enough (some shadows run large). The persona is at most only partly aware of it, and that friction between the controlled face and the uncontrolled truth is where the character gets its flavor. Constitutive: remove it and they are a different person; a Shadow that agrees with the shown self is no Shadow, and a persona without one is only a role. Keep it the persona's own blindness, not the linked position's Loses restated. Link what it hides where that has a file; where it does not, ask whether it should use a khai type.

## Tell

The involuntary physical sign at the point of blindness: at least one, kept few (too many blur it). It lets another persona, and the reader, know something is off, not what. Write it crystal clear against the Action, deliberate move versus involuntary leak, so any reader knows what falls where; if a Tell reads the same as the Action, the Shadow never surfaces. Link any position or piece it involves where that has a file; where it does not, ask whether it should use a khai type.
