# Plot: [title]

## Taxonomy

The group above this plot: the larger telling it sits within. Name it, then ask whether that whole already exists as a Play with its own file. If it does, link it; if it does not, ask whether it should be. This is what the plot is one of, not its own name (the H1 carries that) and not its origin (Owner carries that).

## Owner

- Project: [project]

## Cue

The atmospheric baseline before anything activates: the sediment of history, the climate, the angle of light, the place the plot starts in. If it is something that happens rather than the air already in the room, it is not the Cue. Link the place it draws from where that has a file; where it does not, ask whether it should use a khai type.

## Action

The visible engine: the public vectors each persona openly drives, and where they collide. If the trajectories never meet, nothing forces the system to move; if it is hidden rather than declared, it belongs in Tension. Link the processes, positions, and personas it drives where they have a file; where they do not, ask whether they should use a khai type.

## Stage

The physical deployment: which personas cross the threshold and the pieces they carry (documents, resources, infrastructure). Arrangement, not intent: if it states what the placement means rather than what is set where, it is not the Stage. Link the personas and pieces it deploys where they have a file; where they do not, ask whether they should use a khai type.

## Tension

The invisible engine: the contraband a persona carries alone that makes the Action impossible to complete cleanly, the knowledge, the lie, the inertia. If the Action could run smoothly to its end there is no Tension, and without it the plot is a list of events, not a plot. Link what the persona carries where that has a file; where it does not, ask whether it should use a khai type.
