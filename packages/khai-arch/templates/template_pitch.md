# Pitch: [title]

## Taxonomy

The group directly above: the production or run this pitch sits within. Name it; link it if it has its own file. Not this one's name (the H1) or its origin (Owner).

## Owner

- Project: [project]

## Tenor

The dominant register the run is tuned to: the key its fixed events are played in (grieving, comic, dread, heroic, and so on). One dominant tenor per run; name it plainly. A play may carry a default tenor; the Director may tune it to another.

## Undertow

The counter-current beneath the tenor: the pull against the dominant that keeps the tone from going flat (the laugh inside the grief, the dread under the wonder). What complicates the key without breaking it.

## Nerve

What the pitch is built to press in the audience: the feeling it strikes. Where Tenor names the key, Nerve names the target it plays for.

## Echo

What the pitch leaves once the run is over: the aftertaste that should linger when the events have ended.
