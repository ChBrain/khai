Add the khai-engineer skill. In khai-engineer mode the agent wires a khai engine
into one connected whole: one anchor element that names the engine, every member
tied down from the anchor and back up to it, and siblings tied across to each
other, all woven in prose. Mode A wires a new engine as it is built; Mode B
repairs a flat engine by adding only the missing ties. The skill ships a graph
self-check (anchor-down, member-up, sibling-across, no orphan) so the wiring can
be verified by tool or by hand.
